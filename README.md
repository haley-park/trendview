# TrendView — 데일리 트렌드 뷰어

유튜브·쇼츠·릴스·X(트위터)·스레드·틱톡·**샤오홍슈(小红书)** 인기 콘텐츠와 AI 영상 소식을
카테고리별 × 조회수순으로 매일 확인하는 웹사이트입니다. API 키 없이 무인증 공개 엔드포인트만 사용합니다.

원본 로컬 파이썬 앱(trend-viewer)을 Next.js(App Router)로 포팅해 Vercel에 배포한 버전입니다.

## 기능

- **유튜브 / 쇼츠 탭**: 카테고리(전체·✨추천·AI·먹방·뷰티·브이로그·예능·영화/드라마·테크·지식·여행·중국·해외생활·동물) × 기간(오늘/이번 주/이번 달) 필터, 접이식 정렬 메뉴(조회수/좋아요), 카드 클릭 시 팝업 재생. 유튜브 내부 검색 API(InnerTube)를 언어별 로케일·검색어로 조회수순 호출.
- **✨ 추천 카테고리**: 최근 검색 키워드 + 구독 계정의 관심사 태그를 조합해 추천 영상을 구성 (localStorage 기반, 서버에 개인정보 저장 없음)
- **🤖 AI 영상 탭**: 새 영상 생성 모델·트렌딩 모델(Hugging Face) + 국내·해외 AI 영상 뉴스(구글 뉴스 RSS)
- **📸 릴스 탭**: 인스타그램 웹 내부 API(무인증)로 구독 계정 최신 릴스 수집. 조회수/좋아요/댓글/🚀떠오르는순 정렬. 차단 시 계정 바로가기 폴백
- **𝕏 트위터 탭**: 임베드용 syndication API(무인증)로 최신 트윗+참여수 수집. Vercel IP 차단 시 r.jina.ai 리더 경유 폴백으로 실데이터 유지. 디폴트는 AI 빌더·유명 AI 계정. 좋아요/댓글/리트윗/조회수 정렬
- **🧵 스레드 탭**: 무인증 GraphQL 조회 시도 → 차단 시 Jina 리더가 렌더링한 프로필 마크다운을 파싱해 실제 글+참여수 표시. 디폴트는 AI 빌더 계정
- **🎵 틱톡 탭**: tikwm 무료 공개 API로 실시간 인기 피드(언어별 지역) + 구독 계정 영상. 조회수/좋아요/댓글/🚀떠오르는순 정렬
- **📕 샤오홍슈 탭**: 실시간 인기 검색어 랭킹(60s 무료 공개 API, vvhan 폴백) + ✨관심 키워드 추천(한국·한국여행 고정 + 최근 검색어 + 구독 태그의 中文 변환) + 계정 바로가기. 개별 노트는 서명 API·로그인월로 무인증 조회 불가
- **🚀 떠오르는 계정**: 팔로워는 적지만 게시물당 반응(좋아요+댓글)이 높은 계정을 반응률순으로 표시 (릴스·틱톡)
- **💡 추천 계정**: 구독 중인 계정과 관심사가 겹치는 계정을 추천, + 클릭으로 바로 구독
- **🌐 다국어**: 한국어 / English / 中文 / 日本語 — UI는 물론 유튜브 검색 로케일, 카테고리 검색어, 틱톡 지역, 뉴스 피드까지 언어별로 전환
- **캐시**: 결과는 서버 1시간 캐시, 우측 상단 새로고침 버튼으로 강제 갱신

구독 계정·언어·검색 기록은 브라우저 localStorage에 저장됩니다.

## 개발

```bash
npm install
npm run dev   # http://localhost:3000
```

## 참고 사항

- 인스타그램/스레드는 데이터센터 IP(Vercel)의 무인증 요청을 자주 차단합니다. 차단 시 자동으로 계정 바로가기 카드로 폴백합니다.
- 샤오홍슈는 공식 무인증 API가 없어(서명 x-s/x-t 필요) 무료 공개 프록시(60s API)의 핫서치 랭킹을 사용합니다.
- 틱톡은 tikwm 무료 API 특성상 동시성이 높으면 일시 제한될 수 있어 낮은 동시성(3)으로 호출합니다.
